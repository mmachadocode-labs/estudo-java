# Estudo Java

Site estático para organizar a trilha de estudos **Programação com Base Sólida**.

O objetivo é estudar Java, web, backend e computação aplicada sem recomeçar do zero. Cada módulo tem:

- objetivo;
- checklist;
- conteúdo mínimo;
- prova prática;
- critério de aprovação.

## Módulos

1. Fundamentos de programação
2. Java como ferramenta
3. Orientação a objetos aplicada
4. Collections, Generics e organização de dados
5. Lambdas, Streams e Optional
6. HTTP, APIs e REST
7. Spring Boot básico
8. Banco de dados, JPA e PostgreSQL
9. Validação e tratamento de erros
10. Lombok, records e verbosidade
11. Segurança básica em backend
12. Threads, async e tarefas em segundo plano
13. Testes
14. Projeto final

## Como usar

Abra `index.html` no navegador ou publique com GitHub Pages.

Ritmo sugerido:

- 6 a 8 horas por semana;
- um módulo por vez;
- estudar, praticar, validar e avançar;
- usar o projeto final como consolidação.
